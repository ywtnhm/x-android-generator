# x-android-generator

针对Android项目代码生成工具可在线生成activity、activity.xml、editActivity、edit_activity.xml、adapter、adapter.xml、present、editPresent代码，生成后只须要复制到Android项目对应的包模块中就可以

# 本地部署

下载源码
Eclipse、IDEA运行Application.java，则可启动项目
